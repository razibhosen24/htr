  • The details element

The 'details' specifies additional details that the user
can open and close on demand.
click the triangle
Details

The 'details' element is often used to create
an interactive widget that the user can
open and close. By default, the widget is closed.
When open, it expands and displays the content within.
Any sort of content can be put inside the 'details' element.

  • The del and ins elements

The 'del' defines the text that has been deleted from a document.
Browsers will usually strike a line through the deleted text.
On the other hand, the 'ins' define that text is inserted
and browsers will usually show an underline below the inserted text.
Example : I like coffe not tea

  • The fieldset,label and input elements

Fieldset : The 'fieldset' draws a box around the related elements.

Form : The 'form' is used to create an HTML form for user input.

Input: The 'input' specifies an input field where the user can enter data.

Label: The 'label' defines a label for several elements.
