HTML Tag Reference

  • The bdi element

BDI stands for Bi-Directional Isolation.
The 'bdi' tag isolates a part of the text that might be formatted in a different direction from
other text outside it.

বিডিআই অর্থাৎ বাই ডিরেকশনাল আইসোলেশন এই এলিমেন্ট টি লিখাকে অর্গানাইজ করে।
যেমন ধরুন আমরা বাংলা এবং আরবিতে কয়েকজনের নাম লিখতে চাই। যখন আমরা বাংলা নামগুলো লেখা শুরু করব
তখন বাংলা নাম গুলো বাম দিক থেকে লেখা শুরু হবে কিন্তু আরবি নাম গুলো ডান দিক থেকে লেখা শুরু হয়
যার জন্য আমাদের উল্লেখিত কন্টেন্টের সৌন্দর্য নষ্ট হয়।এই বিডিআই এলিমেন্ট ব্যবহার করে
আমরা প্রত্যেক ভাষার লিখাকে একই ডিরেকশনে লিখতে পারি।

  • The bdo element

BDO stands for Bi-Directional Override.
The 'bdo' is used to override the current text direction.

স্বাভাবিক কোন লেখাকে যদি আমরা উল্টা ভাবে উপস্থাপন করতে চাই সেই ক্ষেত্রে
বিডিও এলিমেন্টি ব্যবহার করতে পারি। যেমন :

  • The button Element

The 'button' defines a clickable button.

(p,h2,h3,ul,li,u,br,strong,b,button,hr)

  • The caption,table,tr,th and td elements

The 'caption' defines a table caption.
It must be inserted immediately after the 'table' element.
using 'tr' for table row
using 'th' for table header
using 'td' for table data
